# TextDad
## What this ?
this is a simple text editor
## Perpubies
- Python3
- Python libraries => qdarkstyle ,PyQt5
## How install Python libraries ?
```
python -m pip install qdarkstyle
python -m pip install pyqt5
```
## Install & Run
```
git clone https://github.com/PooyaRezaee/TextDad.git
cd TextDad
python3 TextDad.py
```
